Introduction

API documentation for Clippy, a simple and elegant URL shortener service.

Base URL: https://clippy.msiamn.dev

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by authenticating with the /api/login endpoint.

Authentication

APIs for managing user authentication

Login user

Authenticate a user and return an access token for API access.

Example request:
curl --request POST \
    "https://clippy.msiamn.dev/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"user@example.com\",
    \"password\": \"password123\"
}"

const url = new URL(
    "https://clippy.msiamn.dev/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "user@example.com",
    "password": "password123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "message": "Successfully logged in",
    "success": true,
    "token": "1|abc123def456ghi789jkl012mno345pqr678stu901vwx234yz",

        "created_at": "2023-01-01T00:00:00.000000Z",

    }
}
